🧠 Pertanyaan 4: Interface Kontrol

Seperti apa tampilan kontrol di web?

A

Slider per Servo

4 slider (0-180°) — satu untuk base, up/down, forward/backward, gripper. Simple, langsung, mudah diimplementasikan. Cocok untuk tugas kuliah.

B

Virtual Joystick + Slider

Joystick untuk gerakan (base rotate + extend), slider untuk up/down + gripper. Lebih intuitif. Sedikit lebih kompleks.

C

Preset Positions + Slider

Tombol preset (ambil, angkat, taruh, home) untuk gerakan cepat, plus slider untuk fine-tuning. Paling fungsional tapi paling kompleks.
