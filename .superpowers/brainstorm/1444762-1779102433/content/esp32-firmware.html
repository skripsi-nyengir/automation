📟 ESP32 Firmware Design

Ringan, sederhana, reconnect otomatis.

Alur Program

setup():
  1. Connect WiFi (STA mode)
  2. Init 4 servo PWM pins
  3. Set all servo ke 90° (home)
  4. Connect ke MQTT broker
  5. Subscribe ke "robot/servo/#"

loop():
  1. client.loop() — jaga koneksi MQTT
  2. Auto-reconnect jika WiFi/MQTT putus

onMessage(topic, payload):
  1. Parse topic → tentukan servo mana
  2. Parse payload → nilai 0-180
  3. Konversi ke pulse width
  4. Tulis ke pin servo

Libraries

WiFi
Built-in ESP32
PubSubClient
MQTT client Arduino
ESP32Servo
PWM servo control

Pin Mapping (contoh)

Servo Base → GPIO 13
Servo Up/Down → GPIO 12
Servo Fwd/Bwd → GPIO 14
Servo Gripper → GPIO 27

Pin bisa kamu sesuaikan dengan wiring-mu.

Topic → Servo Mapping

robot/servo/base → Servo Base (GPIO 13)
robot/servo/updown → Servo Up/Down (GPIO 12)
robot/servo/arm → Servo Forward/Back (GPIO 14)
robot/servo/gripper → Servo Gripper (GPIO 27)
robot/emergency → Stop semua servo (special command)
