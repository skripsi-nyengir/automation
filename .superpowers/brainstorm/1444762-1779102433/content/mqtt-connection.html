🔌 Koneksi Browser → MQTT Broker

3 opsi. Rekomendasi saya ada di nomor 2.

1

Direct WebSocket (ws://host:9001)

Browser connect langsung ke port 9001 Mosquitto.

Simpel, tapi: port beda (9001 vs 8080) → CORS issue di beberapa browser, dan harus tahu hostname server.

Contoh: new MQTT.connect('ws://192.168.1.10:9001')

⭐ 2

Nginx Reverse Proxy (Recommended)

Nginx proxy MQTT WebSocket dari /mqtt → localhost:9001.

Same-origin: browser connect ke ws://localhost:8080/mqtt — port sama, nggak ada CORS. Browser auto-detect hostname dari URL web.

Kode di app.js: const wsUrl = 'ws://'+window.location.hostname+':8080/mqtt'

3

Config File + Input Manual

Punya file config.js dengan broker address. Atau user input IP di halaman setup pertama kali.

Fleksibel, tapi ribet: tiap ganti jaringan harus ganti config.
