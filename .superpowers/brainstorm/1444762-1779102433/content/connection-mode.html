🧠 Pertanyaan 3: Koneksi Web → ESP32

Bagaimana cara web app berkomunikasi dengan ESP32?

A

ESP32 sebagai Access Point (AP Mode)

ESP32 buat WiFi sendiri. Laptop/HP connect langsung ke ESP32. Paling sederhana, tanpa router. Tapi cuma 1 device bisa connect.

B

ESP32 di Jaringan WiFi yang Sama (STA Mode)

ESP32 dan HP/laptop connect ke WiFi router yang sama. Bisa multi-device, range lebih luas, tapi butuh router.

C

Web Bluetooth (BLE)

Akses ESP32 via Bluetooth dari browser. Tanpa WiFi, langsung pairing. Tapi browser support terbatas, agak ribet setup-nya.
