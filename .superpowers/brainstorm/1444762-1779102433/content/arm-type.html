🧠 Pertanyaan 1: Tipe Robot Arm

Robot arm 4 servo — kira-kira ini tipe arm yang seperti apa?

A

Articulated Arm

Seperti lengan robot industrial — base → shoulder → elbow → wrist/gripper. Joint bergerak vertikal & horizontal.

B

SCARA-like / Pick-and-Place

Gerakan horizontal (2 servo) + vertical (1 servo) + gripper (1 servo). Cocok untuk memindahkan barang.

C

Custom / Bebas

4 servo untuk eksperimen, bisa konfigurasi ulang. Fleksibel untuk berbagai project.
