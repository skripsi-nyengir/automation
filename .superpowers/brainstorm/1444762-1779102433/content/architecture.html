🏗️ Proposed Architecture

System Overview
▶ USER
🌐 Browser
└─ MQTT.js → publish ke topic robot/servo/#
⬆️ HTTP (port 8080)  ⬆️⬇️ WebSocket (port 9001)
🐳 DOCKER COMPOSE
└─ nginx:alpine → serve static HTML/CSS/JS (port 8080)
└─ eclipse-mosquitto → MQTT broker (port 1883, 9001)
⬆️⬇️ MQTT TCP (port 1883)
📟 ESP32
└─ WiFi STA → connect ke jaringan yang sama
└─ PubSubClient → subscribe ke topic robot/servo/#
└─ Servo ESP32 library → PWM ke 4 servo
Servo Base Servo Up/Down Servo Fwd/Bwd Servo Gripper

Topic Structure

robot/servo/base → 0-180° (rotasi horizontal)
robot/servo/updown → 0-180° (naik turun)
robot/servo/arm → 0-180° (maju mundur)
robot/servo/gripper → 0-180° (buka tutup)
robot/status ← ESP32 laporan balik (opsional)
